# Roḍah – Plant Store Web 🌿
Roḍah is a clean and modern plant shopping web application that allows users to browse, filter, and purchase indoor and outdoor plants. The app includes a smooth cart experience, real‑time total calculation, a search bar, and a confirmation popup when adding items to the cart.

# Features 🌱
* Category Filtering
 Filter plants by Indoor or Outdoor types.
*  Search Bar
  Quickly find plants by typing keywords.
* Add‑to‑Cart Popup
 - A small popup appears when an item is added.
 -  Includes a success message: “Item added successfully”.
* Dynamic Cart System
  - Automatically calculates the total price.
  - Supports adding multiple items with updated totals.
* Responsive Layout
  Works smoothly on all screen sizes. 

# Technologies Used 🛠️
* HTML5
* CSS3
* JavaScript  (Vanilla JS)
